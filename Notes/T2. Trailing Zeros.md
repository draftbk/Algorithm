# T2. Trailing Zeros

> Easy 一道数学类的题

求一个数阶乘末尾 0 的个数，0 可能是 10 或者 2*5，而 2 肯定比五多，所以其实就是求有多少个 5, 例如:

```
15! 里面有 5，10，15，所以是 3 个 5，是 3 个 0
25! 里面有 5，10，15，20，25，同时 25 算两个5 所以是 6 个零
```
所以 25 算 2 个 5，125 算 3 个 5，然后下面的算法就好理解了。

```
def trailingZeros(self, n):
    if n<0:
        return 0
    result=0
    while n>0:
        n=int(n/5)
        result=result+n
    return result
```
