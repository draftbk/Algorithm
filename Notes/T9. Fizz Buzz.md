# T9. Fizz Buzz

> Easy 

暴力解法，代码一看就能会

```
def fizzBuzz(self, n):
    # write your code here
    result=[]
    for i in range(1,n+1):
        result.append(self.helper(i))
    return result
        
def helper(self,i):
    if i%15==0:
        return "fizz buzz"
    if i%5==0:
        return "buzz"
    if i%3==0:
        return "fizz"
    return str(i)
```
